The Member Details page is a complete profile for a single community member. Access it by clicking any member in the Members table or any username in the Incidents, Chat History, or Overview pages.

<Frame>
  ![Member Detail](/images/member-detail.png)
</Frame>

## Identity and status

At the top of the profile: display name, @username, first seen date, last active timestamp, and current platform.

Below that, the moderation record: warnings, forgiven incidents, mutes, and bans - all time totals.

## Roles

Current roles are shown as badges. Add or remove roles directly from this page using the **\+ Admin**, **\+ Moderator**, and **\+ Whitelist** buttons.

## Moderation actions

Take action on this member without leaving their profile:

| Action | Options |
| --- | --- |
| **Mute** | 1 hour / 6 hours / 24 hours / 3 days / 7 days |
| **Ban** | Permanent (duration support varies by platform) |
| **Unrestrict** | Lifts all active restrictions (unmute \+ unban) |
| **Whitelist user** | Adds whitelist role - bypasses malicious content detection |

## Sentiment and engagement

Four at-a-glance stats:

- **Avg Sentiment** - average positivity score across analyzed messages
- **Impact Score** - a composite measure of the member's influence in the community
- **Reactions** - total reactions received, with average per message
- **Analyzed** - number of messages scored out of total messages sent

**Sentiment over time** shows a 30-day chart of positivity scores. Filter by message type (Questions, Technical, Helpful, Social, Memes, Sarcastic, Swearing) to see sentiment patterns within specific categories.

## Message profile

A breakdown of the member's messages by topic tag - how many were Questions, Social, Technical, and so on. Useful for quickly classifying a user's behavior pattern.

## Community impact and reactions

**Community Impact** shows how this member's activity affects overall community health. **Community Love** shows reactions received - a signal of positive standing in the community.

## Recent messages

The last several messages from this member, with full inline moderation controls (Delete, Mute, Ban) and sentiment tags. Click any message to open its full detail view.

## User Network

Click **See User Network** to view connections between this member and others - useful for identifying coordinated activity or bot networks.
