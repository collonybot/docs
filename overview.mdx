Overview is the first page you land on when you open a project. It gives you a snapshot of community health for the selected time range without having to dig into individual sections.

<Frame>
  ![Overview 1](/images/overview-1.png)
</Frame>

## Sentiment breakdown

The sentiment panel shows the split of Positive, Neutral, and Negative messages for the selected period, with message counts for each. The 24h sentiment delta shows movement since the previous day.

Click **View** to go to the full Sentiment page.

## Key metrics

Six stats across the top:

| Metric | What it shows |
| --- | --- |
| **Members** | Unique members active in this period, with net change |
| **Messages** | Total messages analyzed by the bot |
| **Bot Replies** | How many times the bot responded in chat |
| **Incidents** | Total moderation events, as a percentage of total messages |
| **Actions** | Breakdown of deletions, mutes, and bans |
| **Avg Response** | Bot response latency |

## Quick Settings

Four toggles available directly from Overview without opening Settings:

- Basic chat
- FAQ responses
- Emoji reactions
- Allow bot to ban directly

Changes here apply immediately and sync with the Settings page.

## Schedule status

Shows the current bot mode (Chat \+ Security / Security only / Inactive) and the active time window. Click **View** to open the Schedule page.

## Recent Incidents

The last few moderation events with links to the full incident. Click **See All** to go to the Incidents page.

## Community Highlights

A summary of the last 7 days:

- **Community Loved** - most reacted-on messages
- **Most Active** - members with the highest message count
- **Most Positive Users** - highest average sentiment score
- **Most Negative Users** - lowest average sentiment score

## Recent Chat

A live feed of the most recent messages in the project. Click **View All** to go to Chat History. Inline moderation actions (Delete, Mute, Ban) are available directly from the feed.
