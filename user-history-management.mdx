Manage user moderation actions from one screen.

1. Open **User History** in the setup menu.

![User history management on collony.ai](/public/Screenshot2025-09-22at08.54.57.png)

2. Choose a button:
   - **User Warnings**
   - **Muted Users**
   - **Banned Users**
   - **All Users**
3. Select and search for a user from recent activity.
4. Proceed with your actions.
